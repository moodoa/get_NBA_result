# Get NBA result from ESPN
![image](https://miro.medium.com/max/1050/1*ypslG7x_vZF28O9X-l1ZVQ.jpeg)
## Introduction
### 
get_NBA_result.py
* 以 gameid 為參數爬取 NBA 對戰組合的戰績以及球員數據。
* 回傳值有三項，分別是 1,`分數`、2,`A隊數據`、3,`B隊數據`。
